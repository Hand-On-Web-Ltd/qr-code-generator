# QR Code Generator 📱

A simple, browser-based QR code generator. Type in a URL or any text, get a QR code instantly, download it as PNG.

## Features

- Generates QR codes in real-time as you type
- Download as PNG with one click
- Adjustable size
- Works offline (uses qrcode.js via CDN on first load)
- No tracking, no server, no sign-up

## How to Use

1. Open `index.html` in your browser
2. Type or paste a URL / text
3. Click "Download PNG" to save it

## Use Cases

- Link to your website on business cards
- Add QR codes to flyers, posters, or menus
- Share WiFi passwords
- Link to Google Maps directions
- Quick link to a PDF or document
